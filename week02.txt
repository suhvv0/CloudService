1교시 
- 클라우드 서비스 시스템 소개
- Git bash 설치 (git-scm.com)

2교시
- 기초 리눅스 명령어
ls (폴더 보기 명령)
ls -a (숨긴 파일 포함 폴더 보기 명령)
ls -l (폴더 자세히 보기 명령)
touch 파일명 (파일이 없는 경우 빈 파일 생성)
cat 파일명 (텍스트 파일 CLI 뷰어)
clear (현재 화면 클리어)
cd 폴더명 (현재 폴더에서 지정한 하위 폴더로 이동)
cd .. (현재 폴더에서 바로 상위 폴더로 이동)

- 깃 명령어
git init (현재 폴더를 깃이 관리하는 폴더로 지정. .git 파일이 생성됨)
git status (현재 폴더의 변화를 감지하고 상태를 보고함)
